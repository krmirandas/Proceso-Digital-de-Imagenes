----------------------------------------PARA USAR EL PROGRAMA------------------------------------------

Instalar Python y su sistema de manejo de paquetes
sudo dnf install python
sudo dnf install python-setuptools
sudo dnf install python-pip
sudo pip install --upgrade pip

Instalar los paquetes para buscar archivos
sudo dnf install python-tk

Instalar los paquetes para el proceso de imagenes
sudo pip install pillow

Con esto instalado podemos ejecutar el programa con el comando
python Main.py
